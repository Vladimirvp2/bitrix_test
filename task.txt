Создать свою js-библиотеку {DOCUMENT_ROOT}/bitrix/js/my_lib/ которая:

На странице https://{HOST}/crm/start/ в меню "Ещё" (прикрепленный 
рисунок) удалит пункт "Настройки" и добавит собственный пункт "Мой пункт 
меню" использовав событие "onPopupFirstShow".

При клике на пункт меню отобразить в произвольном виде (например: id, 
название, ответственный сотрудник) информацию о последнем лиде c помощью 
метода
BX.PopupWindowManager. Информация о лиде должна быть запрошена с 
собственного ajax-сервера. Кроме информации о лиде в поп-ап окне должна 
быть кнопка "Закрыть".
Ajax-сервер должен возвращать json-объект:
{
    STATUS: "OK",
    DATE: "{текст}"
}
Для этого создать ajax-сервер {DOCUMENT_ROOT}/ajax/my_server.php, в 
котором вызвать статические метод собственного класса 
\my_module\Myсlass::Mymethod() который вернёт требуемые 
данные(информацию о последнем лиде).
